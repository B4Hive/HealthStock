# HealthStock

Trabalho de Modelagem de Sistemas (DCC117)

## Pré-requisitos

- Java 17 ou superior
- Maven 3.6+ instalado

## Compile/Run

### Opção 1: Usando Spring Boot Maven Plugin

**Linux/macOS:**
```shell
mvn spring-boot:run
```

**Windows:**
```cmd
mvn spring-boot:run
```

### Opção 2: Compilar e executar JAR

**Linux/macOS:**
```shell
mvn clean install && java -jar ./target/HealthStock-1.0-SNAPSHOT.jar
```

**Windows:**
```cmd
mvn clean install && java -jar .\target\HealthStock-1.0-SNAPSHOT.jar
```
